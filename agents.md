# Project Agents & Meta-Documentation

## Overview
This repository contains a high-end, professional photography portfolio for an automotive photographer, designed with an ultra-minimalist, cinematic dark mode aesthetic.

## Development Agent
- **Lead Developer**: Antigravity (Advanced Agentic Coding AI)
- **Framework**: Vanilla (Framework-less)
- **Methodology**: Performance-first, semantic HTML, and custom CSS design systems.

## Design Systems
- **Palette**: Cinematic Dark (`#030303`), Text (`#f5f5f5`), Accent (`#D35400` - Burnt Orange).
- **Typography**: 
  - Serif: *Playfair Display* (Headlines)
  - Sans-Serif: *Inter* (Body & Interface)
- **Interactions**: 
  - `IntersectionObserver` for fade-up micro-animations.
  - Custom CSS-based masonry layout for dynamic galleries.
  - Skeleton-loader-enhanced lazy loading for performance.

## Core Capabilities
1. **Responsive Masonry**: Fluid grid system that adapts to mobile devices by collapsing into a single-column layout.
2. **Project-Specific Storytelling**: Dedicated pages for high-impact visual narratives.
3. **Optimized Asset Loading**: Lazy-loading implementation using `data-src` to ensure fast initial page loads.
4. **Professional Header**: Sticky, semi-transparent navigation with backdrop-blur and "Book a Session" call-to-action.
